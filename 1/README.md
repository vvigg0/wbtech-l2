# Задача l2.1

## Задание
Что выведет программа?

Объясните вывод программы.
```go
package main

import "fmt"

func main() {
  a := [5]int{76, 77, 78, 79, 80}
  var b []int = a[1:4]
  fmt.Println(b)
}
```
## Решение

Вывод будет `[77 78 79]`. Срез `b` содержит в себе элементы массива `a` с индексами 1,2 и 3. (правая граница среза не включается в срез)
